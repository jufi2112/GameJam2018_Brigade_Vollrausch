# General
It's about blocks that hover!
They hover in a not too large map. 

# Specs

The aim is to create a very fast paced (3D) hover block racing game.

**Must have:**
 * hovering blocks
 * a map

**Nice to have:**
 * AI enemies
 * multiple maps
 * breaking blocks

# Art Style

The art style is low poly and flat shaded.

# Maps

 * mostly rural
 * mountain
 * some trees
 * a small see
 * bushes
 * maybe a small city --> buildings
